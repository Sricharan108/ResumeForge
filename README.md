# 🛠️ ResumeForge

**ResumeForge** is a streamlined web application designed to help developers and professionals "forge" high-quality resumes with ease. Stop fighting with Word document margins and focus on your content.

🚀 **[Live Demo](https://sricharan108.github.io/ResumeForge/)**

---

## ✨ Features

* **Real-time Preview:** See your changes instantly as you type.
* **Modern Templates:** Clean, ATS-friendly designs that recruiters love.
* **Markdown Support:** Use familiar syntax to style your professional experience.
* **Export to PDF:** Get a print-ready document in one click.
* **Responsive Design:** Edit your resume from any device.

---

## 🛠️ Installation & Setup

If you want to run ResumeForge locally or contribute to the project, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Sricharan108/ResumeForge.git](https://github.com/Sricharan108/ResumeForge.git)
    ```

2.  **Navigate to the directory:**
    ```bash
    cd ResumeForge
    ```

3.  **Open the project:**
    * If it's a simple HTML project, just open `index.html` in your browser.
    * If it uses a framework (like React or Vue), run:
    ```bash
    npm install
    npm start
    ```

---

## 📖 How to Use

1.  Enter your **Personal Information** (Name, Contact, LinkedIn).
2.  Add your **Work Experience** and **Education** history.
3.  Select your preferred **Template**.
4.  Click **Download PDF** to save your new resume!
